# National Exam Style Questions

Each question has a complete runnable C++17 solution in this folder.

| # | File | Focus |
|---|------|-------|
| 1 | `q01_bst_traversals.cpp` | Build a BST and print inorder, preorder, postorder |
| 2 | `q02_dijkstra_shortest_path.cpp` | Find shortest paths from one source |
| 3 | `q03_undirected_cycle_detection.cpp` | Detect a cycle in an undirected graph |
| 4 | `q04_heap_operations.cpp` | Implement a max heap with insert and delete root |
| 5 | `q05_count_leaf_nodes.cpp` | Count leaf nodes in a binary tree |
| 6 | `q06_topological_sort.cpp` | Order tasks in a directed acyclic graph |
| 7 | `q07_connected_components.cpp` | Count and print graph components |
| 8 | `q08_prim_mst.cpp` | Build a minimum spanning tree using Prim |
| 9 | `q09_kruskal_mst.cpp` | Build a minimum spanning tree using Kruskal |
| 10 | `q10_avl_insert_search.cpp` | Insert and search in an AVL tree |

Compile any file with:

```bash
g++ -std=c++17 file_name.cpp -o program
```
